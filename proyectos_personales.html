En este apartado podemos ver una serie de proyectos que he realizado durante el transcurso de los estudios universitarios que he realizado, tanto en el grado como en el máster, destacando sobretodo aquellos en los cuales he tenido libertad para realizar diferentes modificaciones y extensiones y sobre todo centrándome en aquellos que se podrían considerar como videojuegos

Algunas de las características que podemos encontrar entre estos proyectos son:

Programación en C++, Java, Python y más.

Diseño de juegos y simulaciones interactivas.

Implementación de patrones de diseño y programación dinámica.

Desarrollo de interfaces de usuario con Java Swing y Unreal Engine.

Repositorio de Github

A pesar de que se han incluido los proyectos más llamativos, se puede acceeder al resto de repositorios públicos consultando mi Github personal

Carrera Saludable

Trabajo de Fin de Grado que consiste en un videojuego realizado con Unreal Engine en el cual el escenario y elementos interaccionables se generan de forma pseudoaleatoria para ofrecer una experiencia arcade y en la que el jugador manejará a un personaje que irá obteniendo mejoras y penalizaciones según las decisiones saludables que tome a medida que juega al juego.

BrickBreaker

Videojuego hecho en C++ para la asigantura Software de Entretenimiento y Videojuegos. Clickando en este elemento podremos consultar una presentación con capturas del juego

Videojuego RPG

Videojuego RPG hecho en Python para la asignatura POO utilizando diferentes patrones de diseño y programación dinámica

Space Invaders

Videojuego hecho con Java y Swing simulando un Space Invaders utilizando Drag and Drop para la asignatura Comunicación Persona Máquina

Vaqueros

Videojuego hecho en Unreal Engine para la asignatura Software de Entretenimiento y Videojuegos

Casino Roulette

Aplicación hecha con Java y Swing que simula el funcionamiento de un Casino con su lógica además de incluir una implementación de una tienda de bebidas e intercambio de ficha, implementado para la asignatura Comunicación Persona Máquina

Monopoly

Implementación con una interfaz por consola utilizando Java de una versión del Monopoly, hecho para la asignatura Metodología de la Programación

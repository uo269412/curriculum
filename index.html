Sobre mí

Soy Javier Carrillo González, y esta es mi página personal donde se podrá conocer un poco más mi trayectoria tanto a nivel educativo como a nivel profesional, además de poder acceder a diferentes tipos de proyecto que he realizado accediendo a Github y listando aquellas destrezas que considero bastante relevantes.

Para facilitar la corrección del ejercicio, a pesar de que se puede acceder a todas las páginas utilizando el menú de navegación o clickando en diferentes elementos, aquí listo todas las páginas que contiene este sitio web

Index

Estudios y trabajos

Destrezas
